
## 2024-10-25 - Explicit Control Instructions
**Learning:** Custom canvas apps lack native interaction hints, requiring explicit control instructions on empty/start states (e.g., the splash screen) to improve intuitive usability for first-time users.
**Action:** Add explicit controls (e.g. "Use Left/Right Arrows to move") onto empty or start states when no other visual hints are available in custom rendering applications.
